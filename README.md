# LVZHENHUA'S BLOG

使用 [Hugo](https://gohugo.io/) + [Lvzh 主题](https://github.com/lvzh2012/hugo-theme-lvzh) 构建的个人博客。

## 本地开发

```bash
git clone --recursive git@github.com:lvzh2012/blog.git
cd blog
npm install
hugo server -D
```

访问 http://localhost:1313

## 部署

通过 GitHub Actions 自动部署到 GitHub Pages: https://lvzh2012.github.io
