# 第五章：扩展八股（内存 · RunLoop · 关联知识点）

> **学习位置**：学完 Block（第三章）和多线程（第四章）后阅读本章。RunLoop 把「线程」和「事件源」串起来，为后面第六章、第七章的触摸事件打基础。

| 主题 | 要点 |
|------|------|
| **内存管理** | MRC retain/release/autorelease；ARC 编译器插入 retain/release |
| **weak 原理** | Side table + weak 哈希表；dealloc 时置 nil（与第一章 Non-pointer isa 呼应） |
| **Copy 修饰** | 属性 copy 防 mutable 被改；Block 默认 copy 到堆（见 [第三章 Block](./03-Block面试八股.md)） |
| **atomic vs nonatomic** | atomic 只保证 getter/setter 原子性，不保证线程安全（见 [第四章 多线程 §4.7](./04-多线程面试八股.md)） |
| **RunLoop** | 触摸事件、Timer、Source 都在 RunLoop；主线程默认 RunLoop；子线程需手动 run |
| **AutoreleasePool** | 池底 `AutoreleasePoolPage` 双向链表；RunLoop 休眠/退出时 drain |
| **Block** | 详见 [第三章 Block 面试八股](./03-Block面试八股.md) |
| **多线程** | 详见 [第四章 多线程面试八股](./04-多线程面试八股.md) |
| **performSelector 内存泄漏** | 非 ARC 下 two-arg 版本可能 leak（afterDelay 等） |
