# 第八章：综合串联题（高频）

> **学习位置**：前面 1～7 章学完后做本章，把 isa、Runtime、Block、事件链串成完整面试答案。

## Q1：调用 `[obj method]` 完整过程？

1. 编译为 `objc_msgSend`
2. nil 检查（64 位后部分合并进 msgSend）
3. isa → cache → 类方法列表 → superclass 链
4. 动态解析 `resolveInstanceMethod:`
5. Fast Forwarding → Normal Forwarding
6. `doesNotRecognizeSelector:` 崩溃

---

## Q2：KVO 原理与 isa 的关系？

1. 动态子类 `NSKVONotifying_XXX`
2. `object_setClass` 改 isa 指向子类
3. 子类重写 setter，调用 `will/didChangeValueForKey:`
4. 移除 KVO 时 isa 改回原类

---

## Q3：点击按钮，从触摸到响应全链路？

1. **传递链**：`UIApplication` → `UIWindow` → … → `hitTest` 找到 `UIButton`
2. **响应链**：`touchesBegan` 发给 Button；若 Button 内部手势识别成功，可能 cancel 后续 touch
3. Button 内部 `UIControl` 触发 `sendAction:to:forEvent:` → target-action

---

## Q4：Category 能添加属性吗？原理？

- 不能直接加 ivar（内存布局编译期已定）
- 可以声明 property，Runtime **不会**自动生成 getter/setter
- 用 **Associated Objects** 存值，手动实现 accessor

---

## Q5：+load 和 +initialize 区别？

| | +load | +initialize |
|--|-------|-------------|
| 调用时机 | 类加载时 | 首次发消息时 |
| 调用次数 | 一次 | 每个类最多一次 |
| 继承 | 不继承 | 可继承父类实现 |
| 线程 | 加载线程（可能非主线程） | 首次消息的线程 |
| Category | 各自 +load 都会调 | Category 无 +initialize |
