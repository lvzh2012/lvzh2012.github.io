# 第九章：复习建议

## 推荐复习顺序

按章节编号 **1 → 9** 循序渐进，每一章都建立在前一章的基础上：

| 阶段 | 章节 | 核心目标 |
|------|------|----------|
| **① 对象模型** | 第一章 isa | 能画实例 → 类 → 元类链，理解 Non-pointer isa |
| **② 动态机制** | 第二章 Runtime | 消息发送三步转发，能写 `objc_msgSend` 流程 |
| **③ 闭包与内存** | 第三章 Block | 三种类型、捕获规则、循环引用、`copy` 修饰 |
| **④ 并发** | 第四章 多线程 | GCD 队列/async/sync、死锁、线程安全 |
| **⑤ 桥梁** | 第五章 扩展 | 内存管理、weak、RunLoop（连接线程与事件） |
| **⑥⑦ 触摸事件** | [Common 第六、七章](../Common/README.md) | Hit-Test 自上而下 vs 响应链自下而上，别混 |
| **⑧ 综合** | 第八章 串联题 | 把 isa、Runtime、事件链串成完整答案 |
| **⑨ 收尾** | 本章 | 对照清单查漏补缺，准备项目案例 |

## 必背清单

1. **isa 链**：能画实例 → 类 → 元类 → 根元类，并说出 superclass 方向
2. **消息转发**：三步名称 + 对应方法名，能写伪代码
3. **Block**：栈/堆/Global 三种类型 + 循环引用解法
4. **GCD**：主队列 sync 死锁 + group/once 使用场景
5. **Hit-Test + Responder**：一个自上而下、一个自下而上，别混
6. **结合项目**：准备 1 个 Method Swizzling、1 个 hitTest 改点击区域、1 个 Block 打破循环引用、1 个消息转发的实际例子
