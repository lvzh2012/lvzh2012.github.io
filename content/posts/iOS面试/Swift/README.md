# iOS / Swift 面试八股文

> 按 **循序渐进** 顺序编排：从类型系统 → 方法派发 → 闭包 → 并发 → 语言特性 → 触摸事件 → 综合串联 → 复习收尾。  
> 与 [OC 目录](../OC/README.md) 平级，UIKit / GCD 等共用部分会交叉引用。

```
① 类型系统          ② 派发机制          ③ 闭包              ④ 并发

│ 01 值/引用  │ →  │ 02 派发     │ →  │ 03 闭包     │ →  │ 04 并发     │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘

┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐

```

| 主题 | OC 章节 | Swift 章节 | 说明 |
