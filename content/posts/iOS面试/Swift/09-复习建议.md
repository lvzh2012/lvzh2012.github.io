## 必背清单

1. **值 vs 引用**：struct 拷贝、class 共享、CoW 三句话
2. **四种派发**：static / vtable / witness / message，各什么场景
3. **闭包**：escaping vs non-escaping、`[weak self]` 标准写法
4. **并发**：`@MainActor`、`async/await`、Actor vs lock、Sendable
5. **Optional**：enum 本质、安全解包、`??` 与 chaining
6. **some vs any**：返回值 opaque vs existential 集合
7. **Hit-Test + Responder**：与 OC 相同，别混方向
8. **混编**：`NSObject` + `@objc` + `@objc dynamic`（KVO/Swizzling）

---

## Swift vs OC 对照速查

| 考点 | OC | Swift |
|------|-----|-------|
| 默认类型 | 引用（对象） | 值（struct/enum） |
| 方法调用 | 消息派发 | 静态派发为主 |
| 多态 | 继承 +消息 | 协议 + 泛型 + 继承 |
| 闭包 | Block 三类型 | escaping/non-escaping |
| 并发 | GCD 为主 | async/await + Actor |
| 空值 | nil（指针对象） | Optional enum |
| KVO | 直接支持 | @objc dynamic |

---

## 结合项目准备 4 个案例

1. **值类型建模**：一个用 struct 的 Codable 模型，说明为何不用 class
2. **闭包解循环**：网络层 `[weak self]` 或 async 取消
3. **MainActor UI**：ViewModel 异步拉数据更新 `@Published`
4. **hitTest 扩大点击区** 或 **协议 + 泛型** 封装网络层
