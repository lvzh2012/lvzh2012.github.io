## 使用建议

1. 先过本地 **OC/Swift 1～5 章 + Common 6～10 章**
2. 按目标公司刷对应真题文件
3. 用 [06-高频考点TOP50](./06-高频考点TOP50.md) 查漏补缺
4. 每题准备 **30 秒版 + 3 分钟版** 答案，结合自己的项目

## 算法附录（大厂常考）

| 题型 | 代表题 | 出现公司 |
|------|--------|----------|
| 链表 | 反转、合并有序链表、找环入口 | 字节、网易、腾讯 |
| 二叉树 | 最近公共祖先 LCA | 字节一面 |
| 排序 | 快排/堆排复杂度、TopK | 阿里三面、美团三面 |
| 哈希 | 两数之和、LRU 缓存 | 百度、字节 |
| 字符串 | 最长无重复子串 | 多家 |

**阿里二面场景题**：1G 内存对 1T URL 文件去重 → Bitmap / 外部排序 / 布隆过滤器（客户端岗也可能考）。
